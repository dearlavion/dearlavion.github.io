
"`==` checks **if two references point to the same object**; `equals()` checks **if two objects have the same content**."

## **`==` operator**

### **What**

> Checks **if two references point to the exact same object in memory**.

### **Why**

> Use `==` when you want to know **if two variables refer to the same object**, not whether their content is the same.

### **How / Example**
```
String a = new String("Java");
String b = new String("Java");

System.out.println(a == b);  // false → different objects
System.out.println(a.equals(b));  // true → same content


## Works for primitives
int x = 5;
int y = 5;
System.out.println(x == y); // true → compares values
```



## **`equals()` method**

### **What**

> Checks **if two objects are logically “equal”**, i.e., their content is the same.

### **Why**

> Use `equals()` when you want **value/content comparison** rather than reference comparison.

### **How / Example**
```
String s1 = new String("Java");
String s2 = new String("Java");

System.out.println(s1.equals(s2)); // true → content is same
```

Note: Some classes (like `String`, `Integer`) **override `equals()`** to compare content.  
Custom classes should **override `equals()`** if you want logical equality.

```
class Person {
    String name;
    Person(String name) { this.name = name; }

    @Override
    public boolean equals(Object obj) {
        if(this == obj) return true;
        if(obj == null || getClass() != obj.getClass()) return false;
        return name.equals(((Person)obj).name);
    }
}

Person p1 = new Person("Alice");
Person p2 = new Person("Alice");
System.out.println(p1.equals(p2)); // true → content-based equality
```

 **If you don’t override `equals()`**
- Every class in Java implicitly extends `Object`.    
- `Object` has a default `equals()` method:
```
public boolean equals(Object obj) {
    return this == obj;  // checks reference equality
}
```
That means, by default, `equals()` behaves exactly like `==` — it only checks if the two references point to the same object, not whether their content is equal.

**What happens in your example if you remove `equals()` override ?**
```
class Person {
    String name;
    Person(String name) { this.name = name; }
}

Person p1 = new Person("Alice");
Person p2 = new Person("Alice");

System.out.println(p1.equals(p2)); // false
System.out.println(p1 == p2);      // false

```
