| Operation    | Description                                             | Example                                                      |
| ------------ | ------------------------------------------------------- | ------------------------------------------------------------ |
| **filter**   | Keep elements that match a condition                    | `.filter(u -> u.getAge() > 25)`                              |
| **map**      | Transform elements from one type to another             | `.map(User::getName)`                                        |
| **flatMap**  | Flatten nested structures (`List<List<T>>` → `List<T>`) | `.flatMap(List::stream)`                                     |
| **distinct** | Remove duplicates                                       | `.distinct()`                                                |
| **sorted**   | Sort elements (natural or custom order)                 | `.sorted()` or `.sorted(Comparator.comparing(User::getAge))` |
| **peek**     | Peek into elements for debugging                        | `.peek(System.out::println)`                                 |
| **limit**    | Take only first N elements                              | `.limit(5)`                                                  |
| **skip**     | Skip first N elements                                   | `.skip(3)`                                                   |
## Examples
a) **Filter users by age > 25**
```
List<User> users = List.of(
    new User("Alice", 30),
    new User("Bob", 20),
    new User("Charlie", 28)
);

List<User> adults = users.stream()
                         .filter(u -> u.getAge() > 25)
                         .collect(Collectors.toList());

System.out.println(adults); // Alice, Charlie

```
b) **Map `List<User>` → `List<String>` of names**
```
List<String> names = users.stream()
                          .map(User::getName)
                          .collect(Collectors.toList());

System.out.println(names); // ["Alice", "Bob", "Charlie"]

```
 c) **Flatten a `List<List<String>>` → `List<String>`**
```
List<List<String>> nested = List.of(
    List.of("a", "b"),
    List.of("c", "d"),
    List.of("e")
);

List<String> flat = nested.stream()
                          .flatMap(List::stream)
                          .collect(Collectors.toList());

System.out.println(flat); // [a, b, c, d, e]

```
d) **Sort a list of strings**
```
List<String> unsorted = List.of("Charlie", "Alice", "Bob");

List<String> sorted = unsorted.stream()
                              .sorted() // natural order
                              .collect(Collectors.toList());

System.out.println(sorted); // [Alice, Bob, Charlie]

// Custom order (reverse)
List<String> reverse = unsorted.stream()
                               .sorted(Comparator.reverseOrder())
                               .collect(Collectors.toList());
System.out.println(reverse); // [Charlie, Bob, Alice]

```
e) Use `peek()` to **debug pipelines**:
```
users.stream()
     .filter(u -> u.getAge() > 25)
     .peek(u -> System.out.println("Filtered: " + u.getName()))
     .map(User::getName)
     .forEach(System.out::println);

```
f.)  **skip(n)**
- Skips the **first n elements** of a Stream
- Useful for **pagination**, or ignoring a certain number of elements
```
List<String> names = List.of("Alice", "Bob", "Charlie", "David", "Eve");

// Skip first 2 elements
List<String> skipped = names.stream()
                            .skip(2)
                            .collect(Collectors.toList());

System.out.println(skipped); // [Charlie, David, Eve]

```
g.) **limit(n)
- Takes only the **first n elements** of a Stream
- Useful for **top-N questions**, sampling, or limiting results
```
List<String> names = List.of("Alice", "Bob", "Charlie", "David", "Eve");

// Take first 3 elements
List<String> limited = names.stream()
                            .limit(3)
                            .collect(Collectors.toList());

System.out.println(limited); // [Alice, Bob, Charlie]

```
