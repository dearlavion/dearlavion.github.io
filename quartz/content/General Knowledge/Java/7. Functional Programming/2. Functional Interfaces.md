## What
Functional interfaces are **interfaces with a single abstract method**, which allows us to pass **lambdas** or **method references**.


## Why
Functional interfaces let us use lambdas. You could create your own, but standard ones are preferred because they’re readable, standard, and work directly with Streams.”

```
+----------------+----------------+--------------------------+
| Interface      | Input          | Output                   |
+----------------+----------------+--------------------------+
| Predicate<T>   | T              | boolean                  |
| Function<T,R>  | T              | R                        |
| Consumer<T>    | T              | void                     |
| Supplier<T>    | none           | T                        |
| BinaryOperator | T, T           | T                        |
+----------------+----------------+--------------------------+
```


## 1️⃣ **Predicate <'T'>**
- Represents a **boolean-valued function** (a test/condition)

**Method:**
`boolean test(T t)`

**Example:** Filter users older than 18
```
Predicate<User> isAdult = u -> u.getAge() >= 18;
List<User> adults = users.stream()
                         .filter(isAdult)
                         .collect(Collectors.toList());
```
**Interview Tip:**
- _Used for conditions, usually with `filter`, `anyMatch`, `allMatch`._


## 2️⃣ **Function<T, R>**
- Represents a function that **takes an input T and returns a result R**

**Method:**
`R apply(T t)`

**Example:** Convert Users to their names
```
Function<User, String> getName = User::getName;
List<String> names = users.stream()
                          .map(getName)
                          .collect(Collectors.toList());

```
**Interview Tip:**
- _Used for transformations, usually with `map`._


## 3️⃣ **Consumer<'T'>**
- Represents an **operation that consumes a value but returns nothing**

**Method:**
`void accept(T t)`

**Example:** Print all users
```
Consumer<User> printUser = System.out::println;
users.forEach(printUser);
```
**Interview Tip:**
- _Used when performing actions like printing, logging, or modifying something externally._


## 4️⃣ **Supplier<'T'>**
- Represents a **supplier of results**, i.e., produces a value **without any input**

**Method:**
`T get()`

**Example:** Generate a random number
```
Supplier<Double> random = Math::random;
System.out.println(random.get());
```
**Interview Tip:**
- _Used for lazy value generation or factories._
