### **What**

> Inheritance lets a class reuse code from another class or interface.
> 
> - `extends` → inherit from a parent class
>     
> - `implements` → inherit method signatures from an interface
>     

### **Why**

> To **reuse code**, follow a **contract**, and allow **polymorphism** while keeping design clean and scalable.

### **How**

> - Use `extends` to inherit fields and methods from a parent class.
>     
> - Use `implements` to adopt abstract methods from one or more interfaces.
>

```
class Animal { void eat() {} }
interface Flyable { void fly(); }

class Bird extends Animal implements Flyable {
    public void fly() { System.out.println("Bird flies"); }
}
```
Bird inherits `eat()` from Animal and must implement `fly()` from Flyable.
