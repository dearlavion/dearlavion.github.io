
# About

Repository https://dearlavion.github.io/ is generated by quartz

All obsidian generated notes are maintained in #main branch

Local Test:
	cd quartz/

	yarn run quartz build --serve

# To update deploy gitHub page:

1. Generate files in #main branch first
	cd quartz/
	yarn run quartz build 

2. Generated new files are in /public
	
	cd quartz/public
	git init
	git remote add origin https://github.com/dearlavion/dearlavion.github.io.git
	git checkout -b gh-pages
	git add .
	git commit -m "Update site"
	git push -f origin gh-pages --force
		Note: github might take time to deploy updated page
