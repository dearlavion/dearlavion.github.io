
 RestController is a shortcut for `@Controller + @ResponseBody`. Itâ€™s used to build REST APIs that return data instead of HTML views.

|Annotation|Purpose|Return Type|
|---|---|---|
|`@Controller`|Handles web requests, typically for **views**|Returns **view names** (like JSP/Thymeleaf)|
|`@RestController`|Handles web requests for **REST APIs**|Returns **JSON/XML** (object is serialized automatically)|

#### HTTP Mapping Annotations

|Annotation|HTTP Method|Use Case|
|---|---|---|
|`@GetMapping("/path")`|GET|Retrieve resources|
|`@PostMapping("/path")`|POST|Create new resources|
|`@PutMapping("/path")`|PUT|Update existing resources|
|`@DeleteMapping("/path")`|DELETE|Delete resources|

#### Key Request & Response Annotations
|Annotation|Purpose|
|---|---|
|`@RequestBody`|Bind **HTTP request body** to a Java object|
|`@PathVariable`|Extract **variables from URL path**|
|`@RequestParam`|Extract **query parameters**|
|`ResponseEntity`|Full HTTP response with **status, headers, body**|
